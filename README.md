Este repositorio almacena Notas, páginas y la base documental de los ecosistemas creativos y analíticos de Angel Cruger.

## Estructura del Proyecto
* **Página Principal**: Portal de integración de proyectos (`index.html`).
* **Bitácora**: Archivo dinámico que lee notas en formato Markdown (`.md`) directamente desde este repositorio.
* **Secciones Integradas**:
  * Angel Curiosidades
  * ONESTUDIOS
  * God, Devil and Me (GDM)
  * Red Rainbow
  * MOVAIL
  * Fundación DWS

## Funcionamiento de la Bitácora
La sección de noticias (`bitacora.html`) utiliza JavaScript (`noticias.js`) para conectarse a la API de GitHub, extraer las publicaciones guardadas en la carpeta de noticias y renderizarlas en tiempo real, garantizando un ecosistema fácil de actualizar sin necesidad de bases de datos externas.
